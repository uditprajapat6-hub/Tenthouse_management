# 🏕 Tenthouse Management System

## 📌 Description

A full-stack web application for managing tent bookings, designs, and customer details.

## 🚀 Features

* Booking system
* Design selection
* Customer details form
* Dynamic pricing
* Admin dashboard (optional)

## 🛠 Tech Stack

* Frontend: React.js
* Backend: Node.js + Express
* Database: MongoDB

## 📂 Project Structure

* `/frontend` → React app
* `/backend` → Node.js server

## ▶️ Run Locally

### Backend

cd backend
npm install
npm start

### Frontend

cd frontend
npm install
npm start
